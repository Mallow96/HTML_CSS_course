# HTML & CSS 課程學習紀錄 🎨

本儲存庫展示了我在 HTML 與 CSS 課程中的學習成果。這裡包含了從零開始的網頁結構搭建、樣式美化，以及實用的切版練習與專案作品。

## 📚 學習重點
* **HTML5 語意化**：使用正確的標籤（`<header>`, `<main>`, `<footer>` 等）建構具備 SEO 優勢的網頁。
* **CSS 佈局技術**：
    * **Flexbox**：處理一維空間的排列與對齊。
    * **CSS Grid**：建構複雜的二維網格佈局。
* **原生 CSS 變數 (Custom Properties)**：利用 `--main-color` 等變數提高樣式管理效率。
* **響應式網頁設計 (RWD)**：利用媒體查詢 (`@media`) 確保網頁在各種裝置上皆能完美呈現。
* **CSS 動畫與轉場**：透過 `transition` 與 `@keyframes` 增加介面互動感。

## 📁 目錄結構說明
專案依據練習類型與課程階段分類：
* `project/` - 完整的網頁專案實作。
* `layout-practice/` - 針對特定版面（如導覽列、側邊欄、卡片組）的切版練習。
* `basic-exercise/` - 基礎 CSS 屬性與盒模型 (Box Model) 練習。
